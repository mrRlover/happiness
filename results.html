Hypothesis Confirmation

Time Happy is Money

Now we finally get to the answer the age old question, does money buy happiness? The code chunk below shows the stargazer function from the stargazer package. We use this function to display results from the regression. The beauty of this function is that it captures the summary output of the regression and creates an HTML table, which it then styles into publication quality table.

stargazer(nestedData[[3]], type = "html", style = "qje", dep.var.labels = "GDP per capital", column.labels = c("2015", "2016", "2017", "2018", "2019"))

As you can see from the table below, the answer is a resounding YES!!!! All years have a statistically significant coefficient on GDP per capita. The coefficient is statistically significant at the 1% for all three coefficients. The positive coefficient coefficient shows that an increase in GDP per capita increases the happiness score of the country. So yes, money does buy happiness.
